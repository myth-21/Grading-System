# Grading System

## 📌 Project Overview
The **Grading System** is a web-based application built to automate student performance assessment. The system simplifies grade calculation, report generation, and academic tracking for educational institutions.

This project was created as part of academic and practical skill development in software engineering and web technologies.

---

## 🛠️ Technologies Used
- **Backend:** Java
- **Web Server:** Apache Tomcat
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
